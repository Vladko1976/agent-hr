# Рабочий профиль

## Основные параметры

- **Размер компании**: 500+ сотрудников (крупная организация)
- **Судебная активность**: Часто — ежемесячные споры
- **Тип клиентов**: Внешние клиенты (строительный сектор, тепловые сети, электросети)

## Приоритеты в работе

1. **Точность** — абсолютный приоритет (ошибки дорого стоят)
2. **Оперативность** — нужны быстрые ответы на судебные вопросы
3. **Актуальность** — еженедельные обновления нормативной базы

## Распределение времени

- **Трудовое право (ТК РФ)**: 70%
- **Корпоративное право**: 30%

## Взаимодействие с органами

- **Основной уровень**: Муниципальная власть
- **Федеральные органы**: Практически нет
- **СПС**: Консультант Плюс (основной источник)
