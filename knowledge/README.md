# knowledge/ — База знаний

Здесь хранятся:

- **Шаблоны документов** (приказы, договоры, служебные записки)
- **Примеры исков и возражений**
- **Чек-листы** (увольнение, дисциплинарное взыскание, реструктурирование)
- **Справочные материалы** (сроки, размеры штрафов, формулы расчётов)
- **Практические советы** по наиболее частым вопросам

## Рекомендуемая структура

```
knowledge/
├── templates/               # Шаблоны документов
│   ├── prikazы/             # Приказы по форме Т-1, Т-8 и т.д.
│   ├── trudovie_dogovory/   # Трудовые договоры
│   ├── sluzhebnie_zapiski/  # Служебные записки
│   └── iski/                # Образцы исков и возражений
│
├── checklists/              # Проверочные листы
│   ├── uvolnenie.md
│   ├── disciplinary.md
│   └── restrukcionirovanie.md
│
├── spravka/                 # Справочные материалы
│   ├── srocki.md            # Сроки по ТК РФ
│   ├── razmery_shtrafov.md  # Размеры штрафов
│   └── formuly_raschetov.md # Формулы расчётов
│
└── FAQ/                     # Часто задаваемые вопросы
    ├── trudovoe_pravo.md
    └── korporativnoe_pravo.md
```

Создавайте новые файлы по мере накопления примеров и практики.
